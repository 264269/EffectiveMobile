Программа принимает аргументы в формате [имя_аргумента1] [значение_аргумента1] [имя_аргумента2] [значение_аргумента2]...
Параметры:
	—Район доставки (если не указан, фильтрация по нему не применяется): _cityDistrict
	—Время первой доставки (если не указан, фильтрация по нему не применяется): _firstDeliveryDateTime
	—Файл для логов (по умолчанию ./log/log.txt): _deliveryLog
	—Файл для результата (по умолчанию ./data/result.txt): _deliveryOrder
	—Файл с данными (по умолчанию ./data/orders.txt): _deliveryData
Если в качестве имени файла передаётся не *.txt, используется файл по умолчанию.
